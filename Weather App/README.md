# App del Tiempo Moderna

Una aplicación web avanzada y moderna para consultar el clima en tiempo real, con diseño inspirado en apps reales y mejoras premium.

## Características

- **Búsqueda por ciudad**: Ingresa el nombre de una ciudad para obtener su clima actual.
- **Ubicación actual**: Usa la geolocalización para obtener el clima de tu ubicación.
- **Pronóstico de 5 días**: Ve el pronóstico extendido con animaciones fluidas.
- **Datos detallados**: Temperatura, humedad, viento, sensación térmica, presión y visibilidad.
- **Unidades de temperatura**: Cambia entre Celsius y Fahrenheit.
- **Diseño moderno premium**: Gradientes, sombras, cards elevadas, animaciones y efectos de vidrio.
- **Loading states**: Spinner animado mientras carga la información.
- **Interfaz intuitiva**: Input group con iconos, botones con efectos shimmer, y footer informativo.
- **Responsive**: Funciona perfectamente en dispositivos móviles y de escritorio.

## Cómo usar

1. Abre `index.html` en tu navegador web.
2. Para buscar por ciudad: Escribe el nombre de la ciudad y haz clic en "🔍 Buscar".
3. Para usar tu ubicación: Haz clic en "📍 Usar mi ubicación" (asegúrate de permitir la geolocalización).
4. El clima actual con detalles avanzados y el pronóstico de 5 días se mostrarán automáticamente.

## Configuración de la API

Esta app usa la API de OpenWeatherMap. La API key ya está configurada en `script.js` para demostración.

## Tecnologías usadas

- HTML5
- CSS3 (con Gradients, Shadows, Animations, Backdrop-filter y Grid)
- JavaScript (ES6+)
- API de OpenWeatherMap

## Estructura de archivos

- `index.html`: Estructura de la página con loading y footer.
- `styles.css`: Estilos modernos premium con diseño de app real.
- `script.js`: Lógica de la aplicación con estados de loading.
